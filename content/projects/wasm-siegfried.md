Archives NZ sponsored the development of a WASM build for [Siegfried](https://www.itforarchivists.com/siegfried), the file format identification tool that I created. This means that Siegfried can be run in a web browser,  client-side, and without the need for end-users to install the tool. For more details, see this [LinkedIn post](https://www.linkedin.com/pulse/siegfried-front-end-richard-lehane-dlvaf).
