In 2025, I built [Authority Editor](https://authorityeditor.com) for State Records NSW. This is an editing tool to create, manage, and transform retention and diposal authorities in XML. Authority Editor reimplements an original version of the tool (that I also wrote), that has been in use at State Records for the last sixteen years. The new version of Authority Editor is distributed as both a native Windows application and an online tool. Both are free to install and use.
